Initial Approaches

I started off this project by testing various different approaches to aligning the images. The idea was to see which approaches worked best qualitatively (clearest images, least artifacts) and quantitatively (how long it took to align the images). Since many of these initial approaches were based on a window-search algorithm, they were too slow to run on the .tif files and thus these initial tests were constrained to the .jpg files.
