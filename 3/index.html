Project Description

Throughout this project we will we use Delaunay triangulations and affine warping to smoothly and quickly morph facial structures into each other.

Part 1. Defining Correspondences

To perform the required warping, we first need to define correspondences between any two images we would like to warp. For a smooth warp on a reasonably sized image, 30-45 corresponding points are necessary. We define a "reasonably sized image" as being on the order of hundreds of pixels in either dimension. Before defining correspondences, I ensured that the images were the exact same pixel dimensions, along with having their subjects reasonably well aligned.
